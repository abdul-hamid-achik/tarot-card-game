# Client (Web)

The web client is built with Next.js and React, providing a browser-based interface for the Tarot TCG.

## Technologies
- Next.js 15 with App Router
- React 18 with TypeScript
- Tailwind CSS for styling
- Framer Motion for animations
- Zustand for state management
- Server-Sent Events for real-time updates

## Key Components
- `MainMenu.tsx` - Main game menu interface
- `GameBoard.tsx` - Primary game play area
- `TarotCard.tsx` - Card rendering component
- `MatchmakingOverlay.tsx` - Matchmaking UI

See `apps/web/components/game/` for the full component library.
